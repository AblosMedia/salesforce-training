Bulk-editing Country Way subscriptions

On this page:

  • Mass-updating one field on multiple subscriptions
  • Mass-editing fields on multiple subscriptions

Mass-updating one field on multiple subscriptions

SalesForce can help you mass-update one field for multiple subscriptions. Eg. updating multiple subscriptions' renewal date to all be the same date one year from now.

  1. Click on the "Country Way Subscriptions" tab.
  2. Select a list containing the Subscriptions you want to edit and click "Go".
  3. Select the Subscriptions you want to edit by checking the checkbox next to them.
  4. Click the "Mass Update" button to begin the process.
  5. Following the on-screen instructions, select the field to update (steering clear of any fields you don't recognise), enter the new value for this field, review the changes that will be made, then finally click "Update" to perform the mass-update.
  6. Your Subscriptions will now have that field updated with the value you specified.

Mass-editing fields on multiple subscriptions

SalesForce can help you mass-edit a number of Subscriptions at once. Eg. updating multiple subscriptions' renewal date to a different value for each Subscription.

  1. Click on the "Country Way Subscriptions" tab.
  2. Select a list containing the Subscriptions you want to edit and click "Go".
  3. Select the Subscriptions you want to edit by checking the checkbox next to them.
  4. Click the "Mass Edit" button to begin the process.
  5. A table will appear showing the Subscription IDs and the mass-editable field. Make any alterations you wish, then click "Save". Note that the editable fields available can be changed on request.
  6. Your Subscriptions will now have that field updated with the value you specified.
